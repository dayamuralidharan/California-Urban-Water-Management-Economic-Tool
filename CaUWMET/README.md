# CaUWMET

requires python 3.11

To run:

1. `cd <path to CaUWMET>`
2. `poetry install`
3. `poetry shell & poetry run python main.py`

Add argument `-v` for verbose output (prints the optimizatio progress table).

`main.py` Runs the CaUWMET optimization algorithm for each contractor in a for-loop. This process then results in the optimized model outputs being written to excel spreadsheets. An error log, `CaUWMET.log`, is also created for the purpose of tracking the loop progress and errors. Example `xlsx` files and `log` file are provided in this directory.
